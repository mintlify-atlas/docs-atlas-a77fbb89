OMX is a workflow layer for [OpenAI Codex CLI](https://github.com/openai/codex). It keeps Codex as the execution engine and wraps it with structured workflows, parallel agent teams, persistent state, and runtime monitoring — so you go from idea to completion faster and more reliably.

## The recommended first session

<Steps>
  <Step title="Install">
    ```bash
    npm install -g @openai/codex oh-my-codex
    ```
  </Step>
  <Step title="Set up your project">
    ```bash
    omx setup
    omx doctor
    ```
  </Step>
  <Step title="Launch with full power">
    ```bash
    omx --madmax --high
    ```
  </Step>
  <Step title="Use the canonical workflow inside Codex">
    ```text
    $deep-interview "clarify the authentication change"
    $ralplan "approve the auth plan and review tradeoffs"
    $team 3:executor "execute the approved plan in parallel"
    ```
  </Step>
</Steps>
